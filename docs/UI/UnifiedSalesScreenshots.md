# Unified Sales Flow Screenshots

This page previously referenced screenshots from an older visual design.

The old images have been removed from the repository. Add refreshed quick-sale
and work-order billing screenshots here only after they have been captured from
the current JEronAI Operations interface.

## Desktop: Quick Sale

The quick sale screen creates direct POS sales through the same `Sale`,
`SaleLine`, `Payment`, and inventory services used by work-order billing.

## Mobile: Work Order Billing

The work-order billing screen reviews billable rows, prevents duplicate conversion,
and lets the operator choose the settlement path without treating the work order as
an invoice.
